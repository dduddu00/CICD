EPS 파일을 추출해 보세요

한글 파일 악성코드 분석을 시작하세요

차별화된 자동화, 간단하게 분석하세요!

  • 악성코드를 추출하기 위해서는 다양한 도구가 필요했습니다.
  • 기존에 악성코드를 분석하기 위해서는 파일 구조 확인, 압축 해제, 페이로드 복호화, 쉘 코드 추출 및 저장의 과정을 다양한 도구를 사용하는 불편함이 있었습니다.
  • 이 불편함을 해소하고자 단 한 과정으로 일련의 과정들을 통합하여 한 번의 실행으로 해결할 수 있는 실행파일로 제작하여 배포하게 되었습니다.

Service

한글 파일 기반 EPS 추출

파일 구조 확인

파일 구조를 확인하여 EPS 파일의 여부를 확인할 수 있습니다.

압축 해제

압축되어 있는 스트림 내용을 확인할 수 있도록 압축 해제를 진행합니다.

복호화

PostScript로 암호화되어있는 페이로드를 복호화 합니다.

쉘 코드 추출 및 저장

악성코드 분석에 필요한 쉘 코드를 추출하여 저장합니다.

사용 방법

파일 다운로드

Start Download를 클릭해 실행파일을 다운로드하세요!
다운로드 한 실행파일과 분석할 HWP 파일을 같은 경로에 둡니다.

추출할 HWP 파일명 입력 후 구조 확인

추출하고자 하는 HWP 파일을 입력해 주세요.
여기서 주의할 점은 확장자인 <.hwp>도 모두 적어주어야 합니다!

파일 구조 확인 후 EPS 파일이 존재한다면 "EPS Detected!" 가 출력됩니다.
EPS 파일이 존재하지 않을 시 "EPS Nor Detected!"가 출력됩니다.

복호화 Key 입력

화면에 출력된 복호화 Key를 입력해 주세요.

압축 해제한 파일과 복호화 파일 저장 확인

저장된 EPS 추출 내용을 저장한 <~.hwp.eps> 파일과 쉘 코드를 추출하여 저장한 <~.hwp.txt> 파일이 생성되었는지 확인합니다.
